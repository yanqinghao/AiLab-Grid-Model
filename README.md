# AiLab-Components-Demo

## Install Requirements

``` bash
pip install -r requirements.txt
```

## Run

``` bash
python run.py components.docker.demo.Demo
```

``` bash
python run.py components.stream.demo.Demo
```

## Build

``` bash
bash tools.build.sh
```

## Tips

Demo 中包含
1. 算法组件和服务组件的简单 Demo
2. Docker 镜像构建过程
3. 代码加密混淆
4. 上述各过程中的通用 shell 工具
